Project

01. eXgame App

Aplikasi eXgame merupakan proyek skripsi saya yang dirancang sebagai platform pemasaran produk akun game dengan fitur-fitur inovatif. Fitur pertama memberikan opsi antara Free Ads atau Premium Ads dengan biaya Rp. 10,000 untuk meningkatkan eksposur iklan. Fitur kedua mempermudah pencarian produk berdasarkan jenis game dan range harga. Fitur ketiga memungkinkan pengguna mengubah dan menghapus iklan yang sudah tidak berlaku. Fitur keempat, yang terdapat pada halaman My Profile, memungkinkan pengguna mengelola data iklan yang telah dibuat sebelumnya. Aplikasi ini dikembangkan dengan Kotlin dengan menggunakan teknologi (RecyclerView, Retrofit, Bottom Navigation) dimana aplikasi tersebut berkomunikasi dengan backend yang dibangun menggunakan Hapi JS serta menggunakan layanan web AWS sebagai Web Server.

02. Github App

GitHub App, sebuah aplikasi yang dibuat untuk menyelesaikan tugas dari bootcamp dari IDCAMP dimana aplikasi ini dikembangkan menggunakan bahasa pemrograman Kotlin. Aplikasi ini memanfaatkan API bawaan dari GitHub dan Retrofit untuk berkomunikasi dengan API, memastikan pengalaman pengguna yang lancar dan terhubung dengan sumber daya GitHub. Fitur pencarian yang disediakan memungkinkan pengguna dengan mudah menemukan pengguna yang mereka cari. Dengan menggunakan Recycler View, GitHub App menyajikan informasi pengguna secara terstruktur dan efisien. Selain itu, aplikasi ini memberikan keleluasaan kepada pengguna untuk menyimpan pengguna favorit mereka, yang diimplementasikan melalui penggunaan SQLite/Room untuk manajemen data favorit. Dengan kombinasi fitur-fitur ini, GitHub App memberikan pengalaman pengguna yang intuitif dan personal.

03. Movie Catalog App

Movie Catalog App, hasil dari partisipasi dalam bootcamp IDCAMP, dirancang menggunakan Kotlin untuk mempermudah pengguna dalam mencari informasi film. Aplikasi ini menggunakan API TMDB (The Movie Database) dan Retrofit untuk berkomunikasi dengan API, memastikan akses yang lancar. Fitur penyortiran berdasarkan film terbaru dan terlama disajikan untuk memudahkan pengguna. Dengan teknologi Tab Layout, pengguna dapat beralih antar kategori informasi dengan mudah. Selain itu, pengguna dapat menyimpan film favorit melalui manajemen data SQLite/Room. Movie Catalog App memberikan pengalaman menyeluruh dalam mengeksplorasi dan menyimpan informasi film.
